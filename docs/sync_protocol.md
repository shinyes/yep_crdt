## Protocol Flows

1. Heartbeat: `heartbeat`
2. Incremental data sync: `raw_data`, `raw_delta`, `local_file_chunk`
3. Snapshot fetch RPC: `fetch_raw_request` -> stream of `fetch_raw_response`
   End-of-stream sentinel key: `__fetch_raw_done__`
4. Version sync digest: `version_digest`
5. Manual GC: `gc_prepare` -> `gc_prepare_ack` -> `gc_commit` -> `gc_commit_ack` -> `gc_execute` -> `gc_execute_ack`
6. Manual GC abort path: `gc_abort` -> `gc_abort_ack`
